# Czym jest JavaScript?
- JavaScript (JS) to język programowania używany głównie do tworzenia dynamicznych stron internetowych.
- Można go używać także poza przeglądarką – dzięki środowisku **Node.js*.


# Najważniejsze cechy JS:
- język interpretowany (nie trzeba kompilować),
- język dynamicznie typowany (nie podajesz typu zmiennej)
- działa w przeglądarce (frontedn)
- działa na serwerze (backend - Node.js)

:::info Node.js
Node.js® is:
- a JavaScript runtime environment
    - free,
    - open-source,
    - cross-platform
- lets developers create:
    - servers,
    - web apps,
    - command line tools and scripts.

https://nodejs.org/en

:::

# Uruchomienie skryptu JS
- potrzebny Node.js
- utwórz plik `hello.js`:
```javascript
console.log("Hello, JavaScript")
```

- w terminalu w folderze, gdzie jest plik `hello.js` napisz instrukcję `node hello.js`
