# Funkcja strzałkowa

```javascript
// Zwykła funkcja
function suma(a, b) {
  return a + b;
}

// Funkcja strzałkowa
const suma = (a, b) => a + b;
```

W przypadku funkcji strzałkowych:
- Jeżeli mamy jeden parametr, nawiasy wokół niego mogą zostać pominięte.
- Jeżeli funkcja ma tylko jedno wyrażenie, nie trzeba używać klamr `{}`, ani jawnie pisać `return`.
```javascript
// Funkcja z jednym parametrem
const square = x => x * x;

console.log(square(4)); // 16

```
Jeśli mamy więcej niż jeden parametr, nawiasy muszą zostać użyte:
```javascript
const add = (a, b) => a + b;
console.log(add(3, 4)); // 7

```
