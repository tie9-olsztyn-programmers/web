# Środowisko i pierwszy projekt
## Środowisko programistyczne
- nodeJS
- npm,
- WebStorm JetBrains - IDE

## Stworzenie pierwszego projektu
- uruchom WebStorm
- wybierz projekt Vite, template React
- utwórz projekt
- w konsoli wykonaj npm install
- uruchom aplikację (npm run dev lub przez naciśnięcie klawisza play)

## Struktura projektu
- `src/main.jsx` – punkt wejściowy aplikacji, gdzie renderowany jest główny komponent aplikacji.
- `src/App.jsx`  – główny komponent aplikacji; tutaj możesz rozpocząć tworzenie swojego UI.
