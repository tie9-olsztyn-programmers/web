title: Kalendarz
---


:::info `<input type="date" />`
- To natywny, wbudowany w przeglądarkę wybierak daty (Date Picker).
- Na komputerze: Zwykle rozwija mały kalendarz.
- Na telefonie: Otwiera systemowe kołowrotki (iOS) lub kalendarz pełnoekranowy (Android).


Użytkownik na ekranie widzi datę w formacie lokalnym, np. 10.02.2025 (DD.MM.YYYY).
Ale pod spodem, w kodzie (w e.target.value), przeglądarka zawsze trzyma to w formacie technicznym ISO: "2025-02-10" (YYYY-MM-DD).

:::


## Definicja stanu
`const [selectedDate, setSelectedDate] = useState("");`


## Definicja handlera
```tsx
 function handleDateChange(e: ChangeEvent<HTMLInputElement>) {
        // e.target.value zwróci np. "2025-05-15" (string)
        setSelectedDate(e.target.value);
        console.log(e.target.value)
    }
```


## JSX
```jsx

<br />
<br />

<label>
    Wybierz swoją datę urodzenia:
    <input
        type="date"
        value={selectedDate}
        onChange={handleDateChange}
        // ewentualnie ograniczenia dat
        min={"2001-01-01"}
        max={"2030-12-31"}
        />
</label>
<p>Wybrano datę: {selectedDate}</p>

<button disabled={!hasAgreed || textInput.length < 3} type="submit">Submit</button>
</form>

```
