# Jak powstał Node.js?

- JavaScript pierwotnie powstał, żeby działał tylko w przeglądarce (np. obsługa kliknięć, animacje, walidacja formularzy).
- Ale ktoś wpadł na pomysł: „A co, jeśli moglibyśmy używać JavaScriptu poza przeglądarką, np. na serwerze albo w komputerze?”
- I tak powstał Node.js



# Co to jest Node.js?
- To środowisko uruchomieniowe dla JavaScriptu (runtime environment).
- Oparte na silniku V8 (ten sam, co w Google Chrome).
- Pozwala uruchamiać JS na komputerze lub serwerze, a nie tylko w przeglądarce.

# Co daje Node.js?
- Serwer w JS
    - Możesz napisać własny serwer internetowy (np. API albo stronę) w czystym JavaScript.
    - Popularne frameworki: Express.js, NestJS.
- Dostęp do systemu
    - Możesz czytać i zapisywać pliki (fs),
    - obsługiwać sieć (http, net),
    - pracować z bazami danych.
- Ekosystem NPM
    - Node.js ma olbrzymi system paczek: npm (Node Package Manager).
    - Dzięki temu możesz instalować gotowe biblioteki i narzędzia (np. React, Angular, Webpack).
